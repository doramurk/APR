Program se pokreće pokretanjem main programa u datoteci LAB1.py bez dodatnih argumenata.
Pokretanjem main programa, izvodi se 10 funkcija, koje predstavljaju zadatke, rezultati se ispisuju na konzoli.
Mapa projekta sadrzi i .txt fileove koji predstavljaju ulaze za pojedine zadatke.

Za prevođenje programa korišten je Python 3.6 interpreter.
Korišten je text editor PyCharm.
